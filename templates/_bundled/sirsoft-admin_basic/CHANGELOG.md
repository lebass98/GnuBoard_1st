# Changelog

이 프로젝트의 모든 주요 변경사항을 기록합니다.
형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.1.0/)를 따르며,
[Semantic Versioning](https://semver.org/lang/ko/)을 준수합니다.

## [1.0.1] - 2026-07-08

### Added

- 관리자 사이드바 하단에 "사이트 보기" 링크를 추가했습니다 — 클릭하면 사용자 사이트 홈이 새 탭으로 열립니다.

### Changed

- 첨부파일을 허용 개수보다 많이 선택했을 때의 안내 문구를 "이미지는 최대 N장까지…"에서 "최대 N개까지 첨부할 수 있습니다"로 바꿔, 이미지뿐 아니라 문서 등 일반 파일을 올리는 화면에서도 자연스럽게 표시되도록 했습니다.

### Fixed

- 모바일에서 관리자 메뉴를 열면 메뉴 상단 일부가 상단 헤더에 가려 잘려 보이던 문제를 수정했습니다 — 이제 메뉴가 헤더 아래에서 시작해 첫 항목까지 온전히 보입니다.

## [1.0.0] - 2026-07-01

### Added

- 관리자 화면 상단 헤더에 언어 선택 버튼을 추가했습니다 — 이전에는 프로필 메뉴 안에서만 언어를 바꿀 수 있었으나, 이제 헤더에서 바로 전환할 수 있습니다. 이커머스 모듈이 설치되어 있으면 표시 통화 선택 버튼도 헤더에 함께 나타나, 관리자 화면의 가격 표시 통화를 전환할 수 있습니다.
- 관리자 대시보드의 이커머스 영역(상단 바로가기 버튼과 판매 현황·리뷰·문의 위젯)을 이커머스 모듈이 직접 채우도록 바꿨습니다 — 모듈을 켜면 나타나고 끄면 사라지며, 기존에 고정으로 박혀 있던 예시 숫자·문구가 실제 데이터로 대체됩니다.
- 대시보드 하단 "시스템 리소스(CPU·메모리·디스크)" 위젯을 "최근 알림"(최근 발송된 알림 이력) 위젯으로 교체했습니다.
- 대시보드 하단의 최근 알림·최근 활동·시스템 알림·모듈/플러그인/템플릿 상태 카드를 통일된 회색 카드 스타일로 정리하고, 상단의 시스템 상태 배지를 정리했습니다.
- 대시보드 상단 배지에 설치된 템플릿 수와 활성 언어팩 수를 추가로 표시합니다 (각각 활성/전체).
- 막대 그래프에서 단위·규모가 크게 다른 두 값(예: 판매 수량과 매출액)을 좌·우 양쪽 축으로 나눠 그릴 수 있게 했습니다 — 한쪽 값이 너무 작아 막대가 보이지 않던 문제를 해결합니다.
- 관리자 화면 좌측 메뉴를 상단 메뉴 버튼으로 접고 펼 수 있습니다 — 접으면 본문 영역이 넓어지고, 접은 상태는 새로고침 후에도 유지됩니다.
- 템플릿 정보 모달에 해당 템플릿이 사용하는 외부 리소스(외부 스타일·웹폰트·스크립트) 목록을 표시.

#### 레이아웃 편집기

- 사이드바·푸터·테마 전환 버튼·알림 센터·표 컬럼 선택기·섹션 머리(제목+설명) 컴포넌트를 클릭해 선택하고 속성을 편집할 수 있게 했습니다 — 이전에는 이 컴포넌트들을 클릭해도 선택되지 않거나 속성 편집 패널이 열리지 않았습니다. 섹션 머리 컴포넌트는 요소 추가 팔레트에서 새로 꺼내 화면에 배치할 수도 있습니다.
- 메뉴 아이콘 선택(IconSelect)의 옵션 목록 편집이 내장 기본 아이콘 20종을 시작 목록으로 보여주도록 했습니다 — 이전에는 빈 목록에서 시작해 항목을 1개 추가하면 기본 아이콘 전체가 사라지고 그 1개만 남았습니다. 이제 추가/삭제/순서 변경이 기본 목록 위에서 이루어집니다.
- 페이지 헤더의 "액션 추가" 목록 편집을 제거했습니다 — 페이지 헤더는 직접 배치된 버튼이 있으면 액션 목록을 표시하지 않는데, 모든 화면이 직접 배치 방식을 사용하고 있어 이 편집은 어떤 화면에도 반영되지 않는 항목이었습니다(혼란 방지). 페이지 헤더의 제목·설명 편집과 직접 배치된 버튼의 편집은 종전과 동일합니다.
- 폼 속성 편집을 개선했습니다 — "입력 상자 추가"가 기존 폼 항목과 동일한 "라벨 + 입력칸" 묶음을 추가하고(이전에는 라벨 없는 입력칸 단독이 추가되면서 그 자리가 "컴포넌트 로드 실패"로 깨졌습니다), 각 폼 항목 행에서 라벨과 안내 문구(입력칸의 회색 안내 글)를 각각 한국어/영어/일본어로 편집할 수 있습니다.
- 이 템플릿의 컴포넌트(컨테이너·카드·차트·폼 필드·레이아웃 박스 등)를 드래그로 옮기고 다른 컨테이너 안팎으로 재배치할 수 있도록 편집 표식 연동을 추가했습니다. 이전에는 일부 컨테이너 계열 요소가 편집기에서 선택·드래그 대상으로 잡히지 않아, 그 안의 위젯을 바깥으로 빼낼 수 없었습니다. 관리자 화면(실제 운영 화면)의 모양·동작은 종전과 100% 동일합니다.
- 속성 편집 지원을 추가했습니다 — 요소를 선택하면 정렬·글자 크기·색상·여백·크기 등을 친화적인 컨트롤로 조정하고, 모서리를 드래그해 크기를 직접 바꿀 수 있습니다. 목록/폼 화면의 데이터 있음·없음·검증 실패·저장 중 같은 상태를 미리보기로 전환해 볼 수 있고, 관리자 메뉴·회원·모듈/플러그인 목록 등이 샘플 데이터로 실제와 가깝게 표시됩니다.
- 속성 편집에 스타일 항목을 더 추가했습니다 — 글자가 없는 상자 요소에도 글자색을 지정할 수 있고, 그림자·테두리 모양과 색·모서리 둥글기·투명도·스크롤(자동/숨김/가로·세로 자동)을 설정할 수 있습니다. 글자에는 굵게·기울임·밑줄과 양쪽 정렬, 줄바꿈 설정을 추가했습니다. 표시 권한을 지정할 때 실제 사용 가능한 권한만 목록에 나오고, 각 권한의 식별자를 함께 보여 줘 어떤 권한인지 분명히 알 수 있습니다.
- 화면 상태 미리보기를 더 많은 관리자 화면으로 넓혔습니다 — 환경설정의 모든 탭(메일·업로드·SEO·보안·드라이버·고급·알림 정의·언어팩·본인인증·정보), 역할 폼의 신규/수정/검증 실패, 로그인 실패, 비밀번호 찾기 완료, 비밀번호 재설정의 토큰 정상/오류·검증 실패·완료, 본인인증 단계, 템플릿 목록의 사용자/관리자 탭 화면을 편집기에서 전환해 미리볼 수 있습니다.

### Changed

#### 목록·표 표시

- 목록 표에서 날짜순(또는 특정 항목순) 정렬 시, 값이 비어 있는 행이 정렬 방향과 무관하게 목록 맨 앞으로 끼어들던 문제를 수정했습니다 — 예를 들어 페이지 관리에서 "오래된 발행순"으로 정렬하면 발행일이 없는 항목이 맨 앞으로 올라오던 현상이 사라지고, 서버가 정한 정렬 순서대로 표시됩니다.
- 데이터 표(목록 표)의 각 칸 내용을 위쪽 기준으로 정렬하도록 바꿨습니다 — 한 칸에 여러 줄(예: 가격 아래 외화 환산값)이 들어가 칸 높이가 길어져도, 같은 행의 다른 입력칸·내용이 첫 줄 기준으로 가지런히 정렬됩니다.

#### 화면 외형 정합

- 관리자 카드의 제목·설명과 본문 사이 간격을 한 단계 좁혀 카드 내부가 더 컴팩트하게 보이도록 조정했습니다 — 모든 관리자 카드 (환경설정 / 게시판 폼 / 페이지 폼 / 역할 폼 / 회원 폼 등) 에 일괄 반영됩니다.
- 알림 템플릿 / 본인인증 메시지 정의·템플릿 / 언어팩 설치 모달의 입력 행에서 라벨과 입력칸 사이에 옅은 회색 분리선과 세로 간격이 끼어 보이던 회귀를 수정했습니다 — 라벨과 입력칸이 한 필드로 시각적으로 응집되어 다른 관리자 모달의 입력 행과 동일한 결로 보입니다.
- 환경설정의 고급·보안·본인인증 탭에서 카드들이 세로로 나열될 때 카드 사이 간격이 의도보다 부풀어 마지막 카드 아래에 빈 공간이 끼어 보이던 회귀를 수정했습니다 — 카드 묶음 사이 간격이 다른 관리자 화면과 동일한 결로 정합됩니다.
- 환경설정 본인인증 탭의 "목적별 프로바이더 매핑" 과 "등록된 프로바이더 상세" 영역에서 같은 식별자가 페이지 검사 시 여러 번 나타나던 회귀를 수정했습니다 (HTML 표준 위반 해소).
- 환경설정 본인인증 탭의 "목적별 프로바이더 매핑"에서 저장 검증에 실패하면 문제가 된 입력칸이 빨갛게 강조되고 그 아래에 사유가 표시되도록 보완했습니다 — 기본 설정 영역과 동일하게 어느 항목을 고쳐야 하는지 한눈에 확인할 수 있습니다.
- 회원 등록 / 수정 화면을 환경설정 화면과 동일한 시각 결로 정합했습니다 — 화면 폭에 따라 페이지 좌우 여백이 단계적으로 커지고, 상단 탭 내비의 풀-bleed 영역이 페이지 여백 분기와 정확히 맞춰지고, 각 섹션의 헤더가 아이콘 없이 제목만 표시되며, 폼 행 묶음 사이에 옅은 회색 분리선이 자동 적용됩니다. 다른 관리자 화면과 같은 결로 통일됩니다.
- 카드 제목 우측에 부속 요소(예: 확장 출처 배지) 를 정렬하는 표준 시맨틱을 추가 — 제목과 부속 요소를 한 줄에 배치하면서 부속 요소만 우측 끝으로 정렬할 수 있도록 정리.
- 회원 등록 / 수정 화면에서 상단 탭 내비와 본문 콘텐츠가 너무 붙어 보이던 회귀를 수정했습니다 — 탭 내비와 첫 카드 사이에 표준 상단 여백이 자동으로 들어가 환경설정 화면과 동일한 결로 통일됩니다.
- 역할 등록 / 수정 화면의 카드 헤더 정렬과 기본정보 카드 본문의 폼 행 묶음을 다른 관리자 화면과 동일한 결로 정합했습니다 — 헤더 아이콘이 제거되고, 권한 설정 카드의 선택 카운트 배지가 카드 제목 우측 끝으로 정렬되며, 기본정보 카드의 항목 사이에 옅은 회색 분리선이 자동으로 적용됩니다.
- 관리자 화면 전반의 입력 박스·드롭다운·텍스트 영역·태그 입력·상태 배지 외형을 표준화했습니다 — 패딩·배경·모서리·포커스 효과가 화면마다 다르게 보이던 차이가 사라지고, 다국어 입력 탭, 비활성·읽기전용 입력, 옵션 없는 드롭다운까지 동일한 모양으로 표시됩니다. 잠긴 드롭다운이 일반 드롭다운보다 작게 보이던 현상과, 식별자·참조용 읽기전용 입력은 옅은 회색 글자로 "잠긴 정보"임을 구분하도록 정리했습니다.
- 알림 발송 이력 / 본인인증 이력 페이지의 상단 탭을 다른 관리자 화면과 동일한 표준 탭으로 통일 — 색상·모양이 일관되고, 스크롤 시 상단 고정과 모바일 드롭다운 자동 전환이 함께 적용됩니다.
- 사용자 등록/수정·역할 등록/수정 화면의 상단 영역을 재배치 — 페이지 제목은 일반 흐름, 탭만 상단 고정, 취소·저장 버튼은 화면 하단에 항상 고정되어, 긴 폼을 스크롤하는 동안에도 저장 버튼이 항상 보입니다.
- 관리자 화면 전반(환경설정·사용자 폼·역할 폼·본인인증 정책·모달·대시보드·목록 패널)의 카드·입력 칸·라벨·보조 설명·에러 메시지·본문 텍스트·배지·아이콘 버튼·보조 버튼·구분선·여백·2단 그리드 모양을 한 가지 기준으로 통일했습니다 — 화면마다 조금씩 다르게 보이던 카드 모양·padding·간격, 입력 칸 테두리·배경·포커스, 입력 행 사이 구분선, 글자 톤, 새로고침 등 아이콘 버튼·보조 버튼의 다크 모드 외형·포커스 링이 모든 관리자 화면(코어·모듈·플러그인 공통)에서 동일한 결로 표시됩니다. 입력 칸·드롭다운에 외형 클래스가 중복 적용되던 결함과, 입력 행 묶음의 간격이 어색하게 벌어지던 회귀도 함께 해소되었습니다.

#### 대시보드·차트·구조

- 관리자 대시보드 반응형 레이아웃과 시각 구조를 개편.
