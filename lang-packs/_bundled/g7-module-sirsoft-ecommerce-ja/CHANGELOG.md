## [1.0.0] - 2026-07-01

### Added

- 주문 상태 전이 차단 안내 메시지(`validation.orders.status_transition.invalid` / `.bulk_invalid`) 일본어 번역 추가 — 허용되지 않는 주문 상태 변경(단건·일괄·상품옵션 일괄) 시 차단 사유가 일본어 로케일에서 자연스럽게 표시됩니다.
- 관리자 상품 목록 필터 검증 메시지(`validation.list.*` — 카테고리/카테고리 미등록/날짜 유형/판매상태/전시상태/브랜드/브랜드 미등록/과세여부/가격 유형/가격·재고 범위/배송정책) 일본어 번역 추가 — 상품 목록 필터 검증 실패 시 안내 메시지가 일본어 로케일에서 자연스럽게 표시됩니다.
- 배송정책 국가별 설정 검증 메시지(`shipping_policy.country_settings.*` — 국가/배송방법/통화/부과정책/기본배송비/무료배송기준/구간값/계산 API 주소·필수·참고필드) 및 구간별 배송비 음수·필수 안내, 리뷰 설정·배송 계산 API 참고필드 라벨, 상품정보제공고시 활성 상태 변경·이커머스 설정 저장 활동로그, 상품 배송정책 비활성 안내 일본어 번역 추가.
- 쿠폰 1인 사용 횟수 제한 초과 안내(`messages.coupon.per_user_limit_exceeded`)와 관리자 주문설정의 결제수단별 마일리지 차감 시점 라벨·비활성 안내(`admin.settings.payment_methods.mileage_deduction_timing`, `..._disabled_hint`) 일본어 번역 추가 — 쿠폰 사용 한도 초과 안내와 결제수단별 마일리지 차감 시점 설정이 일본어 로케일에서 자연스럽게 표시됩니다.
- 상품 폼·목록·설정의 신규 안내 일본어 번역 추가 — 상품 이미지 개수 상한 초과 안내(`exceptions.product_image_limit_exceeded`), 추가옵션 전체 삭제 확인·외화 자동 표시 안내, 기타정보 라벨 추가·할당 해제·삭제·등록/수정·기간 입력 라벨, 일괄변경 빈 항목·실패 안내, 통화 표시 기호 라벨이 일본어 로케일에서 자연스럽게 표시됩니다.
- 추가옵션 직접입력·검증 안내 일본어 번역 추가 — 추가옵션 직접입력 필수 안내(`exceptions.additional_option_custom_text_required`), 추가옵션 선택 오류·필수 선택 안내(`exceptions.additional_option_invalid`, `additional_option_required`), 추가옵션 그룹·선택지 등록 검증 메시지(그룹명/선택지 최소·최대·필수, 선택지명, 추가금 안내), 관리자 상품 폼의 선택지명·추가료·기본·사용·직접입력 허용 라벨이 일본어 로케일에서 자연스럽게 표시됩니다.
- 통화별 기준 단위(base_unit) 설정 안내·검증 메시지와 미지원 결제 통화 차단 안내(`exceptions.unsupported_payment_currency`) 일본어 번역 추가 — 환율·통화 설정 화면의 "기준 단위" 라벨·안내·검증 실패 메시지와 지원하지 않는 결제 통화 차단 안내가 일본어 로케일에서 자연스럽게 표시됩니다.
- 회원 기본 배송국 설정·국제 배송 주소·배송 가능 여부 안내 일본어 번역 추가 — 회원 배송국 조회/변경 안내와 검증·활동로그(`messages.user_shipping_country.*`, `validation.user_shipping_country.*`, `activity_log.action.user_shipping_country_change`), 배송 불가 상품 안내(`exceptions.country_not_shippable`), 관리자 주문 폼의 국제 주소 입력 안내(주소·도시·도/주·우편번호 placeholder, 휴대폰/전화 필수 안내), 통화·배송국 라벨이 일본어 로케일에서 자연스럽게 표시됩니다.
- 통화별 사용 언어 설정 검증 라벨(`validation.attributes.language_currency.currencies.*.locales`) 일본어 번역 추가 — 환율·통화 설정 화면에서 통화별 "사용 언어" 항목 입력값 검증 실패 시 친화 필드명이 일본어 로케일에서 자연스럽게 표시됩니다.
- 배송정책 통화 고정 안내 문구(`shipping_policy.form.currency_code_fixed_hint`) 일본어 번역 추가 — 배송정책 등록·수정 화면에서 통화가 상점 기본 통화로 자동 적용된다는 안내가 일본어 로케일에서 자연스럽게 표시됩니다.
- 관리자 주문 상세의 결제 통화 청구 안내(`order_detail.payment_currency`, `order_detail.charged_in_payment_currency`) 일본어 번역 추가 — 기본 통화와 다른 통화로 결제한 주문의 청구 통화·금액이 일본어 로케일에서 표시됩니다.
- 레이아웃 편집기 결제(PG) 진입 동작 라벨(`editor.action.request_pg_payment.*`, `editor.action_chip.response_pg_*`) 일본어 번역 추가 — 편집기 동작 추가 목록의 "결제 진입"과 결제 핸들러·데이터 입력 라벨, 결제 응답 데이터 칩 후보 명칭이 일본어 로케일에서 자연스럽게 표시됩니다.

### Changed

- 통화 표기 일본어 번역의 한자 표기를 정정 — 일부 안내에서 중국어 간체 "货" 로 표기되던 "通貨" 를 일본어 정자로 바로잡아 통화 관련 안내(기본/결제 통화 설정·검증·잠금 안내)가 올바르게 표시됩니다.
- 부분취소 차단 안내 메시지(`exceptions.cancel_refund_negative`) 일본어 번역을 쿠폰 단정 없이 일반화된 문구로 갱신 — 할인 조건 변경으로 결제금액이 늘어 취소 불가한 상황이 일본어 로케일에서 정확히 표시됩니다.
- 위안화(CNY) 가격 접두 기호(`messages.currency.prefix.CNY`)를 ¥ 에서 元 로 변경 — 엔화(¥)와 위안화가 같은 기호로 표시되어 혼동되던 문제를 해소해, 장바구니·상품·주문 화면에서 두 통화가 일본어 로케일에서도 구분되어 표시됩니다.
