# Changelog

이 언어팩의 모든 주요 변경사항을 기록합니다.
형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.1.0/)를 따르며,
[Semantic Versioning](https://semver.org/lang/ko/)을 준수합니다.

## [1.0.1] - 2026-07-08

### Added

- 관리자 게시글 목록의 "사용자 화면 보기" 링크 라벨(`view_user_page`) 일본어 번역 추가 — 관리자에서 해당 게시글의 사용자 화면으로 이동하는 링크가 일본어 로케일에서 자연스럽게 표시됩니다.

## [1.0.0] - 2026-07-01

### Added

- 게시판 첨부파일 다운로드 활동이력의 신규 다국어 키(다운로드 액션 라벨·설명)에 대응하는 일본어 번역 추가
- 게시판 모듈 1.0.0-beta.6 의 신규 다국어 키(상대 시간 표현, 관리자 대시보드 게시판 영역 라벨)에 대응하는 일본어 번역 추가
- 레이아웃 편집기 데이터 소스 라벨 일본어 번역 추가 (`editor.data_source.*`) — 편집기에서 게시판 데이터 소스의 표시 명칭이 일본어 로케일에서 자연스럽게 표시됩니다.
- 레이아웃 편집기 검색엔진 설정의 게시글 SEO 자동값 연결 칩 라벨 일본어 번역 추가 (`seo.auto_value.*`) — 게시글 제목·대표 이미지 등 확장이 제공하는 SEO 데이터의 출처 이름이 일본어 로케일에서 자연스럽게 표시됩니다.
- 게시판 복사 안내 창의 신규 안내 문구(복사 대상·제외 항목·새 슬러그 확인)와 "관리자 메뉴에 표시" 필드 라벨의 일본어 번역 추가 — 게시판 복사·검증 화면이 일본어 로케일에서 자연스럽게 표시됩니다.
- 게시판 분류 검증 메시지(분류 최대 개수 초과·빈 분류명)의 일본어 번역 추가 — 분류 추가 제한 안내가 일본어 로케일에서 자연스럽게 표시됩니다.
- 게시판 권한 설정의 "비밀글 조회"(`posts.read-secret`) 권한 이름 일본어 번역 추가 — 권한 안내 메시지에서 권한 이름이 일본어 로케일로 자연스럽게 표시됩니다.
- 게시판 설정 일괄 적용 중단(전체 롤백) 안내 문구와 활동 이력 라벨의 일본어 번역 추가 — 일괄 적용 실패 시 롤백 안내가 일본어 로케일에서 자연스럽게 표시됩니다.
- 첨부 허용 확장자 검증 메시지(최소 1개 필수)의 일본어 번역 추가와 허용 확장자 안내 문구의 일본어 번역 갱신 — 빈 값 저장 차단 정책 변경에 맞춰 일본어 로케일에서 자연스럽게 표시됩니다.

### Changed

- 게시판 금지어(blocked_keywords) 관련 라벨·안내 문구의 일본어 번역을 "禁止ワード" 로 통일 — 게시판 설정·검증 화면의 금지어 용어가 일본어 로케일에서 일관되게 표시됩니다.
- 회원 알림 설정 탭 라벨의 일본어 번역을 "掲示板通知設定" 로 명확화 — 관리자 회원 상세·수정 화면에서 게시판 알림 설정 영역의 명칭이 일본어 로케일에서 다른 알림 영역과 구분되어 표시됩니다.
