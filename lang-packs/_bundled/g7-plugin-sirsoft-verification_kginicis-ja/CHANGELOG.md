# Changelog

이 언어팩의 모든 주요 변경사항을 기록합니다.
형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.1.0/)를 따르며,
[Semantic Versioning](https://semver.org/lang/ko/)을 준수합니다.

## [1.0.1] - 2026-07-09

### Changed

- 라이브 가맹점 MID 프리픽스 표기를 이니시스 정책 변경에 맞춰 SRB 로 갱신했습니다 (라이브 MID 라벨·입력 안내 문구).

## [1.0.0] - 2026-07-01

### Added

- 미성년자 차단 안내와 기타 본인인증 오류 안내의 일본어 번역 추가
- 필수 신원 정보 누락 시 안내(본인인증 정보 불완전)의 일본어 번역 추가
- 본인인증 정보 저장 실패 안내의 일본어 번역 추가
- 라이브 자격증명 항목 이름(라이브 MID·라이브 API 키)의 일본어 번역 추가 — 설정 저장 검증 오류 안내에서 항목 이름이 일본어 로케일로 표시됩니다.
- 레이아웃 편집기 데이터 소스 라벨 일본어 번역 추가 (`editor.data_source.{inicisRecord,settings}`) — 편집기에서 본인인증 기록·설정 데이터 소스의 표시 명칭이 일본어 로케일에서 자연스럽게 표시됩니다.
- 본인인증 채널 라벨(`messages.channels.ipin` = アイピン) 일본어 번역 추가 — 본인인증 이력 화면 등에서 KG이니시스 아이핀 채널이 일본어 로케일에서 자연스럽게 표시됩니다.
- KG이니시스 본인인증 플러그인(sirsoft-verification_kginicis)의 일본어 번들 언어팩 초기 제공
